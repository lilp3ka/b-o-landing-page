# B&O Landing Page

![HTML5](https://img.shields.io/badge/HTML5-orange)
![SCSS](https://img.shields.io/badge/SCSS-pink)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)

A responsive landing page for the B&O brand built with HTML, SCSS, and JavaScript. The project recreates a Figma design with a focus on pixel-perfect implementation, semantic HTML, responsive layouts, and clean code architecture.

## 🚀 Live Demo

[View Live Demo](https://lilp3ka.github.io/b-o-landing-page/)


## 🛠️ Technologies Used

- HTML5
- SCSS
- JavaScript (ES6+)
- BEM Methodology
- Parcel
- Responsive Design
- CSS Flexbox
- CSS Grid

## ✨ Features

- 📱 Fully responsive layout
- 🎯 Pixel-perfect implementation based on the Figma design
- 🧩 Semantic HTML markup
- 🎨 Smooth CSS transitions and hover animations
- 📂 Well-structured SCSS architecture using the BEM methodology
- 🌍 Cross-browser compatibility

## 🏁 Getting Started

Clone the repository:

```bash
git clone https://github.com/lilp3ka/b-o-landing-page.git
cd b-o-landing-page
```

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm start
```

Build the project:

```bash
npm run build
```
